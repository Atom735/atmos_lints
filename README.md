# Atmos Lints Package

## How to use

Add dependency to `pubspec.yaml`

```yaml
dev_dependencies:
  atmos_lints: ^1.0.0
```

Include full lints in `analysis_options.yaml`

```yaml
include: package:atmos_lints/full.yaml
```
